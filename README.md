# MovieCinemaClub

MovieCinemaClub is a review forums app. You have to register in this app to publish review, to vote and comment user's review (like/unlike). Users can also create private group where share news and polls (In this moment only creators of group can share news and create polls).

# Getting Started
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

## Prerequisites
python== 3.11 and django==4.4

## Installing

```
git clone https://github.com/imbimboluciano/ForumCinema.git
```

## Run Virtual Env

Install pipenv (on Windows add pipenv to PATH)

```
pip install pipenv
```
Start pipenv shell and run this command

```
pipenv shell
pip install -r requirements.txt
```

## Start Application

Start pipenv shell and run these command

```
cd ForumCinema
python manage.py runserver
```
